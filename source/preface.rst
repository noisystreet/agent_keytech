为什么需要 Agent？
========================

大语言模型（如 GPT-4、Claude、Llama 3）的出现，让机器具备了前所未有的语言理解和生成能力。但单纯的 LLM 存在三个天然局限：

- **无法获取实时信息**：训练数据存在时间截断，无法回答"今天天气如何"
- **无法执行具体操作**：只能生成文本，不能发送邮件、查询数据库、调用 API
- **无法自主学习**：每次对话都是独立推理，不会从错误中积累经验

**Agent** 正是在这些局限之上构建的智能体框架——赋予 LLM 工具、记忆和规划能力，让它从"会说话的模型"进化为"会行动的智能体"。

============

阅读本书前，建议读者具备：

1. Python 编程基础（本书示例使用 Python + 主流 Agent 框架）
2. 对大语言模型的基本概念有了解（Transformer、Prompt、Token 等）
3. 对 HTTP API、JSON 等 Web 基础协议有所了解
