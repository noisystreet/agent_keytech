===============================
术语表
===============================

.. glossary::

   Agent
       能够感知环境、做出决策并采取行动以实现目标的智能实体。在 LLM 语境中，
       Agent 指代使用大模型进行推理和决策的系统。

   LLM
       Large Language Model，大语言模型。基于 Transformer 架构，通过海量文本
       预训练获得语言能力的神经网络模型。

   RAG
       Retrieval-Augmented Generation，检索增强生成。在 LLM 生成回答前从外部
       知识库检索相关信息的技术。

   ReAct
       Reasoning + Acting，推理与行动交替进行的 Agent 决策模式。

   CoT
       Chain-of-Thought，思维链。引导 LLM 逐步推理的技术。

   Fine-tuning
       微调，在预训练模型基础上用特定领域数据继续训练的过程。

   RLHF
       Reinforcement Learning from Human Feedback，基于人类反馈的强化学习。

   Quantization
       量化，将模型权重从高精度降低到低精度的压缩技术。

   Token
       LLM 处理文本的基本单元，可以是一个字、一个词或一个子词。

   Context Window
       上下文窗口，LLM 一次能处理的 token 最大数量。

   Hallucination
       幻觉，LLM 生成不真实或不合逻辑内容的倾向。

   Embedding
       嵌入，将文本转换为向量表示的数值化过程。
