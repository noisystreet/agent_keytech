# 大模型 Agent 核心技术

[![CI](https://github.com/noisystreet/llm_agent/actions/workflows/ci.yml/badge.svg)](https://github.com/noisystreet/llm_agent/actions/workflows/ci.yml)

大模型 Agent 核心技术入门与进阶教程，涵盖 LLM 基础、Agent 框架、推理能力、多智能体、RAG、安全、部署和进阶技术。

## 文档

在线文档（Read the Docs）：

> **<https://agent-keytech.readthedocs.io/zh-cn/latest/>**

## 本地构建

```bash
pip install -r requirements.txt
make html       # 构建 HTML
make serve      # 构建并在 localhost:8000 启动预览
```

## 目录结构

```
source/

├── chapter_02_agent_framework/ # Agent 框架

├── chapter_04_multi_agent/     # 多智能体系统
├── chapter_05_rag/             # RAG 系统
├── chapter_06_safety/          # 安全与对齐
├── chapter_07_deployment/      # 部署实践
├── chapter_08_advanced/        # 进阶技术
├── examples/                   # 代码示例
└── appendix/                   # 附录
```
